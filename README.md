-->Windows Defender Security & Malware Protection Project

This project demonstrates how Windows Defender protects Windows systems from viruses and malware using different scan types and PowerShell commands.

1.Tools Used
- Windows Defender (built-in antivirus)
- PowerShell
- Excel

2.Steps Performed
- Enabled and updated Windows Defender
- Ran Quick, Full, Customised scans
- Checked Protection History
- Used PowerShell for scan automation
- Documented results and generated report

3.Files Included
- windows_defender_report.pdf (full documentation)
- scan_log.xlsx (scan summary)
- screenshots (Defender scan screenshots)

4.Outcome
- Learned how to detect and remove threats
- Understood Defender settings & PowerShell automation
- Gained practical IT support and security maintenance skills
